# Nodex -- Simple Overview

## What is this?

Nodex is a powerful note-taking app that helps you **organize everything
in one place**---notes, tasks, documents, media, and ideas.

It grows with you over time. And you can make the application grow too.

------------------------------------------------------------------------

## What makes it different?

Instead of keeping notes separate, Nodex lets you **connect everything
together**.

-   Link notes to each other
-   Organize ideas like a tree or a web
-   See how everything relates
-   Extend the app with plugins and scripts

------------------------------------------------------------------------

## What can you store?

You can create many types of notes:

-   Text notes
-   Tasks and to-dos
-   Documents and PDFs
-   Images and videos
-   Drawings and diagrams
-   Code snippets (if needed)
-   Plugins and scripts

Everything can be previewed inside the app.

------------------------------------------------------------------------

## Easy to Use

-   Simple and clean interface
-   Focus on writing and organizing
-   No technical knowledge needed
-   Use plugins from market to extend functionality

------------------------------------------------------------------------

## Grows With You

You can install extra features (plugins) anytime:

-   Add new note types
-   Add new tools
-   Customize how things work
-   Write scripts to automate workflows

You don't need to build anything---just install and use.

------------------------------------------------------------------------

## Example

Create a "Project" note and:

-   Link all related notes
-   Attach files and media
-   Track tasks
-   Keep everything in one place

------------------------------------------------------------------------

## Why use it?

-   Keep everything organized
-   Never lose context
-   Build your own knowledge system over time
-   Tailor the app to your needs

------------------------------------------------------------------------

Nodex is like a **second brain**---a place where your knowledge lives,
