# Nodex Personas – Programmable Knowledge OS

## 🧠 Nodex Reframed

**Nodex = A programmable, versioned, extensible knowledge OS where documents are typed, behavior is scriptable, and UI is composable**

Core primitives:
- Plugins → extend capability  
- Scripting → automate thinking  
- Version control → evolve ideas  
- Typed documents → structured cognition  
- Centralized knowledge → single source of truth  
- Journal system → temporal thinking  
- Composable UI (widgets) → build custom workflows  

---

### 🧠 Mindset
“My knowledge should behave like my codebase.”

### 🔥 What they build in Nodex
- Custom “Architecture Note” document type  
- Plugin to auto-link commits → docs → incidents  
- Script to parse logs → generate incident reports  
- Widget: Live system diagram  

### ⚙️ Feature Mapping
- Plugins → Dev tooling  
- Scripting → automation logic  
- Version control → trace decisions  
- Typed docs → ADR, RFC  

### 💡 Breakthrough
“My documentation executes.”
