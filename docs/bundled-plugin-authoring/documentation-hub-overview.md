# Documentation

Use the **sidebar** (Docs — search):

- **Guides** — bundled read-only Markdown: **User guide** (using Nodex) and **Plugin authoring** (developer topics).
- **Commands** — searchable command API (metadata from registered commands).
- **Settings** — keyboard listing, `window.nodex.shell` shape, **About**; short **User guide** / **Plugin authoring** summaries open as Markdown in the **main area**.

Source files ship under `docs/bundled-plugin-authoring/` in the repository and are **seeded into your workspace notes** when you open a project or when the API server bootstraps the workspace.

The **companion** column shows a **Documentation outline** (table of contents) for the page in the main area. This overview is also a bundled note.
