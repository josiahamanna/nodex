# Application Overview

## Introduction

This application is a flexible, extensible note-taking application designed to
handle diverse content types while enabling deep interconnections
between notes. It combines simplicity for everyday users with powerful
customization capabilities for developers.

------------------------------------------------------------------------

## Core Philosophy

Traditional note-taking tools treat notes as isolated entities. This application
redefines this by allowing notes to exist within a **graph-based
structure**, where any note can link to any other.

This enables users to: - Build knowledge systems instead of static
notes - Organize information contextually - Navigate relationships
naturally

------------------------------------------------------------------------

## Key Features

### 1. Multi-Type Notes

This application supports a wide range of note formats, including: - Text and rich
documents - Code snippets - Drawings (Excalidraw, draw.io) - Images,
videos, and PDFs - Tasks and calendars - Widgets and custom content
blocks

These note types are not limited to built-in options and can be extended
via plugins.

------------------------------------------------------------------------

### 2. Graph-Based Linking

-   Any note can link to any other note
-   Supports both hierarchical (tree) and relational (graph) structures
-   Enables bidirectional navigation and contextual connections

------------------------------------------------------------------------

### 3. Plugin-Driven Architecture

-   New note types can be added via plugins
-   Features can be extended without modifying the core system
-   Encourages a modular ecosystem

------------------------------------------------------------------------

### 4. Scripting & Developer Support

-   Built-in terminal for development workflows
-   Support for scripting to customize behavior
-   Developers can build plugins directly within the app

------------------------------------------------------------------------

### 5. User Experience

#### For Non-Technical Users

-   Install plugins easily
-   Focus on note-taking without complexity
-   Clean and distraction-free interface

#### For Technical Users

-   Full extensibility through plugins and scripts
-   Ability to define custom workflows and UI components

------------------------------------------------------------------------

## Example Use Case: Project Knowledge Hub

A user creates a central note called **"Project X"**.

Within this note, they can: - Link all related documents - Attach and
preview PDFs, images, and videos - Connect tasks and meeting notes -
Store code snippets relevant to the project

This creates a fully connected workspace where all project-related
information is easily accessible and navigable.

------------------------------------------------------------------------

## Vision

This application aims to become a unified workspace where: - Notes are dynamic and
interconnected - Content is not limited by format - Users can scale from
simple note-taking to complex knowledge systems

------------------------------------------------------------------------

## Conclusion

This application bridges the gap between simplicity and power. It provides an
intuitive interface for everyday users while offering deep customization
and extensibility for advanced users, making it a versatile platform for
knowledge management.
